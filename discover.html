Question-1: What are the different ways to select an element in the DOM?

Answer: getElementsByTagName(), getElementsByClassName(), getElementById(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: innerHTML, innerText, and textContent are all JavaScript properties used to interact with the content of HTML elements, but they differ in how they handle HTML tags, whitespace, and CSS styling.

Question-3: What is event delegation in the DOM?

Answer: Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to manage events triggered by its child elements.

Question-4: What is event bubbling in the DOM?

Answer: Event Bubbling: After reaching the target element (child), the event enters the bubbling phase, propagating back up through the DOM tree to the parent (parent).

Question-5: How do you create, add, and remove elements using JavaScript?

The document.createElement() method is used to create a new HTML element. Once an element is created, it needs to be added to the DOM to be visible on the webpage. This is usually done by appending it to an existing parent element. Elements can be removed from the DOM using methods on their parent node or directly on the element itself.
